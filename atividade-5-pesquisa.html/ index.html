Elementos HTML <b>

O elemento <b> é usado para deixar o texto em negrito. Ele serve para dar destaque visual a uma parte do texto, mas sem alterar o significado da frase.

Exemplo de uso:

        <p>Esse é um texto com <b>negrito</b>.</p>

Resultado: Esse é um texto com negrito.
