Elementos HTML <strong>

O elemento <strong> é usado para deixar o texto em negrito e indicar que aquela parte do texto é importante. Ele é parecido com o <b>, mas tem um significado semântico: mostra que o conteúdo tem ênfase.

Exemplo de uso:

        <p>Atenção: <strong>Não esqueça de salvar o arquivo!</strong></p>

Resultado: Atenção: Não esqueça de salvar o arquivo!
